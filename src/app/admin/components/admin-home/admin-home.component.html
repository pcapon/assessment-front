Bienvenue sur l'espace administrateur d'Alto

Cet espace est destiné aux employés d'Alto


Il est important de comprendre que cet admin évoluera en fonction des retours que vous ferez. De plus, il est open-source, ce qui signifie que n'importe qui peut contribuer à son developpement sur Github

Fonctionnement / Impersonnation

Pour simplifier le temps de dev/maintenance et pour garder les même identifiants, il a été choisi de ne pas contruire de backend dédié. L'iconvenient de ceci est qu'on se base sur le même système d'authentification que notre app Alto. Ainsi, les données sont cloisonnées par entreprise pour des raisons de sécurité forte. En tant qu'admin alto, il faut donc d'abord "impersoner" quelqu'un d'une entreprise avant de pouvoir consulter et éditer cette entreprise
